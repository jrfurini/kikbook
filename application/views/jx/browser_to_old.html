Para utilizar o Kik Book você precisa atualizar o seu navegador da internet.


Se quiser fazer isso agora, basta selecionar um dos navegadores abaixo:


Google Chrome;

Firefox;

Opera;

Apple Safari;

Internet Explorer 9;

Internet Explorer 8;

Só utilize a versão 8 do Internet Explorer se você estiver no Windows XP ou Vista. Se estiver no Windows 7 ou Windows 8 selecione o IE 9.




O Kik Book foi desenvolvido com a última tecnologia WEB e utliza HTML5 e CSS3. Infelizmente o seu navegador não suporta este tipo de tecnologia. A atualização vai beneficiar a sua navegação na internet em quase todos os sites.
